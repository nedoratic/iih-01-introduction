Introduction

Setup

  • Open the code editor
  • Open the web browser
  • Keep the code editor and the web browser side by side

Basic workflow

  • Enter some code in the code editor
  • Go from the code editor to the web browser with command + tab
  • Refresh the page in the web browser with command + r
  • Go back form the web browser to the code editor with command + tab
  • Repeat

Basic file manipulation in VSC

  • Create a new file with control + option + command + n
  • Move between open files with control + tab or with option + command + left/right
  • Search through the file command + f
  • Save the file command + s
  • Close the file command + w
  • Close the project command + shift + w

Terminal in VSC

  • Open the terminal with control + tilde
  • Open the file.extension in the web browser form terminal with $ open file.extension
